# Intelligent Traffic System Backend

This is the backend for the Intelligent Traffic System, deployed on Hugging Face Spaces using Docker.
It provides APIs for:
- ANPR (Automatic Number Plate Recognition)
- ATCC (Automatic Traffic Counter and Classifier)
- Accident Detection
